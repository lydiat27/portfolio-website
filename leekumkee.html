Lee Kum Kee Website Redesign
CONTEXT
School Project
ROLE
Interface Design
TOOLS USED
Figma
Problem
The current Lee Kum Kee website feels outdated and lacks a clear visual hierarchy, making it difficult for users to navigate and discover products efficiently. I decided that a redesign was needed to improve usability, modernize the interface, and better showcase the brand's global identity.
Process
  • I started off by creating a site map of the original website, to me it was too cluttered and hard to navigate
  • By figuring out what I wanted to remove, I started piecing together how I wanted the homepage to look
  • I made sure to keep all the pages simple and aesthetically pleasing
  • A lot of trials and errors went into this process but, ultimately I finalized my designs
Solution
Key Takeaway
This redesign demonstrates how improving visual hierarchy and usability can transform a traditional website into a more engaging and intuitive user experience.
